HTML4 pour les Nuls

Vous lisez actuellement HTML4 pour les Nuls, un ouvrage grace auquel vous allez découvrir le monde merveilleux du Web en devenant, vous aussi, auteur. Ces pages vous apporteront leur aide dans ces trois domaines importants :

  • Retrouver des informations sur tout ce qui concerne HTML
  • Avoir sous la main des exemples concrets pour chaque balise.
  • Etre plus facile a consulter que la specification officielle du W3C.
